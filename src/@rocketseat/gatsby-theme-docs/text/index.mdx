# Introduction

The goal is to have a in depth review of react codebase performance, cross checking domain specific best practices
and making sure the right set of standards are used.


## Features

- 📝 Code review
- 🛣 Understanding how system and parts work
- 🖥 Code standards crosscheck according to system needs (React perf and practices)
- 📱 Analyze bundle sizes and distribution
- 📈 API and network load
- 📄 Suggestions and improvements
- 🚦 Internal infrastructure analysis
- 📲 External API usage improvements
